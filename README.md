# edutech-nihongo

Generated by `bootstrapping-project.sh`.

## Packages
- `client/` — React + Vite (SPA)
- `server/app-server/` — Express (Node.js)
- `server/ai-server/` — FastAPI (Python)

## Layout vs Cursor rules
All frontend code and FE config live under `client/` only (no `src/` or Tailwind at repo root). Scaffolds follow: `fe--frontend-structure` (client), `be--nodejs-structure` (`server/app-server/`), `backend-fastapi-structure` (`server/ai-server/`). See `README.structure.md` under each backend package.

## Node + pnpm (per package)
- `client/.nvmrc` and `server/app-server/.nvmrc` can differ — run `nvm use` inside each folder before `pnpm install`.
- No monorepo workspace: each package has its own `pnpm-lock.yaml`.
